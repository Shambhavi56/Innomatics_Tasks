## 📌 Project
FastAPI Internship Final Project  
Project #5 — Movie Ticket Booking System

## 🚀 Features Implemented

### ✅ Beginner (Q1–Q5)
- Home route
- Get all movies
- Get movie by ID
- Movies summary endpoint

### ✅ Easy (Q6–Q10)
- Pydantic validation
- Booking system
- Helper functions
- Promo code logic
- Filter endpoint

### ✅ Medium (Q11–Q15)
- Add movie (POST)
- Update movie (PUT)
- Delete movie (DELETE with business rule)
- Seat hold system
- Seat confirm & release workflow

### ✅ Hard (Q16–Q20)
- Search (multi-field, case-insensitive)
- Sorting (validated fields)
- Pagination (correct total_pages)
- Booking search/sort/pagination
- Combined browse endpoint
